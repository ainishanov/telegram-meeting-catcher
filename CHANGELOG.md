# Changelog

## v0.1.0 - 2026-06-19

Initial public release.

### Added

- Telegram MTProto listener for allowlisted chats.
- Dry-run scan for recent Telegram messages.
- Deterministic meeting extraction for English and Russian examples.
- Optional OpenAI-compatible LLM fallback.
- Google Calendar event creation.
- Calendar dedupe by Telegram source message.
- Local pending review queue for uncertain events.
- Telegram and Google setup scripts.
- Doctor command for environment validation.
- Docker personal instance.
- GitHub Actions CI.
- Privacy, security, runbook, deploy, and personal cloud docs.
