# Launch Posts

Use one simple message everywhere:

Private Telegram meetings become calendar events.

## Show HN

```text
Show HN: Telegram Meeting Catcher - turn Telegram plans into Calendar events

I built a self-hosted tool that watches only the Telegram chats you allow,
catches clear meeting plans, and creates Google Calendar events.

It starts in dry-run mode, has a local review queue for uncertain events, and
dedupes events by Telegram source message.

Repo: https://github.com/ainishanov/telegram-meeting-catcher
```
