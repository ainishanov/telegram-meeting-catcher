# Roadmap

## Now

- Keep the self-hosted path simple.
- Improve deterministic extraction with real anonymized examples.
- Make Docker personal instances reliable.

## Next

- Web setup wizard.
- Calendar event preview before live writes.
- Health endpoint with redacted status.
- Deploy templates for personal cloud providers.
- Encrypted secret store for hosted control plane.

## Later

- Managed personal cloud with one isolated worker per user.
- Billing and account deletion flows.
- Team-level deployment without shared raw chat streams.
